# Expense Tracker

A command-line expense and income tracking application built with Python and Pandas.

## Features

-  Add and track income and expenses
-  Categorize transactions
-  Filter by date range and category
-  Edit and remove transactions


## Requirements

- Python 3.7+
- pandas

## Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/expense-tracker.git
cd expense-tracker
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the application:
```bash
python expense_tracker.py
```

### Main Menu Options:
1. **Add Item** - Add income or expense
2. **View Item** - View and filter transactions
3. **Exit** - Close the application

### Features:
- **Filter by Category**: View transactions by specific category
- **Filter by Date**: View transactions within a date range
- **Sort**: Sort by date or amount
- **Edit**: Modify existing transactions
- **Remove**: Delete transactions

## Data Storage

All transactions are stored in `ExpenseTrackerData.csv` in the same directory.
